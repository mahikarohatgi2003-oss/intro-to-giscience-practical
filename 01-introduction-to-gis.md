# Chapter 1 — Introduction to GIS

## 1.1 Introduction to GIS

Geographic Information Science is the science that underlies Geographic Information Systems (GIS) technology.

- **ESRI** defines GIS as an organized collection of computer hardware, software, geographic data, and personnel designed to efficiently capture, store, update, manipulate, analyse, and display geographically referenced information.
- **Duecker** defines GIS as a special case of information systems where the database consists of observations on spatially distributed features, activities, or events, definable in space as points, lines, or areas. A GIS manipulates data about these points, lines, or areas to retrieve data for ad hoc queries and analyses.
- **USGS** defines GIS as a computer hardware and software system designed to collect, manage, analyse, and display geographically (spatially) referenced data — a comprehensive definition well suited to agricultural applications of GIS.

## 1.2 Evolution of GIScience

Roger Tomlinson introduced GIS in his 1962 report on computer mapping and analysis to the National Land Inventory, Canada Department of Agriculture. Before computer-based GIS technology, oral traditions and maps were the primary means of communicating geographic information. Today, GIS technologies span mapping, spatial analysis and modeling, location-based services, cyber-geographical applications, and spatial crowdsourcing — with related names including Geospatial Science, Spatial Science, Spatial Information Science, Geoinformatics, and Geomatics.

**Timeline of key developments:**

| Period | Milestone |
|---|---|
| 1960s | Line printers used for map representations; Roger Tomlinson leads development of the Canadian Geographic Information System (CGIS), the foundation many consider modern GIS to be built on. CGIS introduced the layer-based approach to managing maps. |
| 1982 | ESRI (now the largest GIS software provider) introduces the Arc-INFO minicomputer program. |
| 1990–2010 | GIS is gradually introduced into schools and businesses; software matures to handle both vector and raster data; growth of satellites expands access to GNSS and space-based GIS data. |
| Present | Open-source boom — free open datasets, satellite imagery at multiple resolutions, and LiDAR data are widely accessible. Online GIS databases (ArcGIS Online, QGIS Cloud) store spatial data at scale; success now depends on quality control and customization to specific needs. |

## 1.3 Components of GIS

1. **Hardware** — CPU, keyboard, visual display unit, printers, plotters, etc. The CPU is the main computing unit; printers/plotters serve as map output devices.
2. **Software** — Programs that run GIS functionality: data input/editing, storage and management, analysis, output, and the user interface.

   ```
   INPUT → STORAGE → ANALYSE → OUTPUT → USER INTERFACE
   ```

3. **Data** — The most important component. GIS integrates both spatial and non-spatial data, and its ability to handle many data types is a key reason for its popularity.
4. **People** — Technical users (managers, data administrators, system operators) and viewers/general users (businessmen, planners, scientists, etc.).
5. **Procedures and Methods** — The management framework governing how data is collected, stored, analysed, and how problems are resolved.
